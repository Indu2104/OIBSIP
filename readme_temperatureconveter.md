Temperature Converter

A simple web-based tool to convert temperatures between Celsius, Fahrenheit, and Kelvin.

Features

Convert temperatures easily.

User-friendly interface.

Built with HTML, CSS, and JavaScript.
How to Use

Open OIBSIP_Webdevelopment_task3.html in a browser.

Enter a temperature.

Select the units to convert from and to.

Click Convert to see the result.
